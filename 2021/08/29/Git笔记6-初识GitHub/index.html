Git笔记6-初识GitHub

Aug 29, 2021

41: GitHub的价值

此前 程序员之间进行编程协作的方式很少
需要项目管理员许可才能fork分支，等待时间长，效率低
Git引入fork概念，开源项目变得容易。但是不能帮助优秀的开源项目传播

GitHub：创建了一个可以托管整个代码的地方，程序员可以协同工作，了解如何充分利用Git
协作和编写软件更容易
更好更直观的版本控制系统

42: GitHub的核心功能

github.com/features

code review/项目管理/集成/团队管理/开源/文档/存放代码
无缝code review
将代码、需求、项目进度集成
看板功能
issuu 需求、任务的管控，通过标签管理
GitHub的集成是第三方提供的
star
market place 依赖管理 部署管理 安全性管理
