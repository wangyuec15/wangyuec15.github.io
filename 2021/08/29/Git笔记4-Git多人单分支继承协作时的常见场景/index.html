Git笔记4-Git多人单分支继承协作时的常见场景

Aug 29, 2021

34: 不同人修改了不同文件如何处理？

远端改变之后，git push会报错

可以顺利的merge

35: 不同人修改了同文件的不同区域如何处理？

没有ahead和behind就是一样的

36: 不同人修改了同文件的同一区域如何处理？

git status发现merge没有完成

37: 同时变更了文件名和文件内容如何处理？

变更文件名的先提交了，后提交的报错

直接检测出文件名的变化，本地的文件名已经变化，自己编辑过的文件也被保存

38: 把同一文件改成不同文件名如何处理？

多人修改同一文件的文件名

发现没有不同
git status是再两边都删除原文件，添加新文件
协商决定采用一种修改之后

冲突解决了还需要commit merge
