cin/cout和scanf/printf的差别，前者是物件，后者是函数。

scanf

int scanf(const char *format, ...)
format 说明符形式为:
[=%[*][width][modifiers]type=]

  • * 从流 stream 中读取的数据不存储在对应的参数中。
  • modifiers h:短（+整型/无符号整型）l:长（+整型/无符号整型）/双精度型 L:长双精度型
  • type %a:一个浮点值 %c:单个字符 %d:十进制整数 %e%f%g:浮点数 %i:十/八/十六进制整数 %o:八进制整数 %s:字符串 %u:无符号十进制整数 %x:十六进制整数 %p:指针 %[]:字符集合 %%:‘%’号

    scanf(“%d%d%d”,&a,&b,&c);

  1. &a、&b、&c 中的 & 是地址运算符，分别获得这三个变量的内存地址。
  2. %d%d%d 是按十进值格式输入三个数值。输入时，在两个数据之间可以用一个或多个空格、tab 键、回车键分隔。

数组元素在内存空间中是连续的
sizeof()输出的是占用的内存空间，如一个int是4个字节；
直接cout<<数组名 是数组的首地址
cout<<数组名[0] 第一个元素的地址，和数组的首地址相同
数组名是常量，不可以进行赋值的操作
访问具体元素的地址 要加取址符“&”
